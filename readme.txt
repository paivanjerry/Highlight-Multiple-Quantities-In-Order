Highlight Multiple Quantities In Order makes important highlight on order if customer have ordered over one quantity of a single product.

== Description ==

This plugin is best fitted for not so precise people. When products are sold mostly with one piece per order, it is easy to overlook multi-quantity orders. This plugin will highlight the background of the quantity cell if the quantity is over one.
